# EDA Histogram App

## Description
An interactive Exploratory Data Analysis app built with Streamlit that allows users to upload any CSV file and explore the data visually.

## Tools Used
- Python
- Pandas
- Matplotlib
- Streamlit

## How to Run
1. Install the required libraries: pip install streamlit pandas matplotlib
2. Run the app: streamlit run app.py
3. Upload any CSV file to explore the data
